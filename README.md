# Web Deb Practice - Landing Page & Steam Store

The purpose of this activity is to use AI tools to assist in web development and content creation.

## Notes
For this example, Vite + React was used to build the example page, we encourage using the same technologies for simplicity, but you can use whatever frontend framework and build tool you like, just consider that the provided GitHub Pages workflow *may*  have to be modified to accomodate for a different setup.

### GH Pages setup
Just make sure that the source for the Build and deployment, under the repository settings is set to github actions.

When selecting the creation of a new workflow, you may paste the contents of the provided ```static.yml`` for quick setup.


### Repository
Creating a repository from scratch is encouraged, but not required, you can fork this repo if you want.
